# Welcome to Upwork'd.AI
### Making your Freelance Job Search Instantaneous!!!

> ### Feature:
> - Job Fitness Score
> - Expected Salary
> - Cover Letter
> - Experience Gaps 
> - Day to Day Activity

1. Enter OpenAI API Token
2. Upload Resume
